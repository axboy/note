安装

  • 下载

https://golang.org/dl/

  • 安装
linux/mac
tar -C /usr/local -xzf go1.4.linux-amd64.tar.gz
export PATH=$PATH:/usr/local/go/bin

mac
直接用pkg安装包，安装路径为/usr/local/go/bin/
brew install go / golang
brew info go / golang
